Rectangle ColliderComponent::getBoundingBox ( )
inline

Get the Bounding Box of all Collider objects associated with this ColliderComponent.

All Collider objects added with addCollider() are user to calculate the bounding box. The bounding box is a Rectangle of smallest dimensions that fit all Collider objects within it.

Returns
Rectangle with format (x1, y1, x2, y2) instead of (x, y, w, h).
