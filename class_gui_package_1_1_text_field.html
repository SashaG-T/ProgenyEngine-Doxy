Detailed Description

A GameObject used to read in user keyboard input.

The TextField will read in user keyboard input when it is active. The TextField can become active either with setActive(bool) or with the user clicking within the TextField's visible boundry. The TextField will become inactive and no longer accept input when clicked outside of the boundary or of course when setActive(bool) is called.

NOTE: Make sure to set STRMAP["GuiPackage.PNG_textField"] = "path/to/image.png" Where *"path/to/image.png"* is the path to a PNG file of an (tileSize*3)x(tileSize*3) px image. The PNG image is 9 sections. Each section is (tileSize)x(tileSize) pixels. Each section represents one of the edges or corners of the TextField. tileSize can be set within the TextField constructor.

Warning
This class is a bit old. It was ported over from an older version for the convenience of debugging and has not been throughly tested but seems to work fine.
